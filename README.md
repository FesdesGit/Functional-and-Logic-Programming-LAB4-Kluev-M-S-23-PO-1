Симулятор парковки (вариант 6)

Описание

Программа моделирует работу парковки с платной стоянкой.
Пользователь может:
1)заезжать на парковку (автоматически назначается свободное место);
2)выезжать (стоимость рассчитывается исходя из времени пребывания);
3)сообщить о потере билета (фиксированный штраф);
4)переключать время (вручную указывать, сколько часов прошло).

Все события логируются и сохраняются в файл parking.log при завершении.

После запуска программа последовательно запрашивает:

1. Номер машины – любая непустая строка.
2. Действие – одна из команд:

Команды:
enter Заезд на парковку. Машина занимает первое свободное место. Время въезда фиксируется как текущее (симулированное) время.
exit Выезд. Стоимость = ceil(текущее_время - время_въезда) * тариф (50 руб/час).
lost Потеря билета. Штраф = 200 руб. Машина удаляется с парковки.
next Переключить время. Запрашивается положительное число часов (например, 1.5). К текущему времени прибавляется указанное значение.
quit Завершить работу. Все логи сессии сохраняются в файл parking.log.

Пример работы:

```
Введите номер машины: A123
Действие (enter/exit/lost/next/quit): enter
Въезд: A123
Машина A123 заехала на место 1 в момент 0.0. Свободно мест: 2

Введите номер машины: next
Действие (enter/exit/lost/next/quit): next
На сколько часов увеличить время? (например 1.5): 2.3
Переключение времени на +2.3
Время изменено на +2.3. Текущее время: 2.3

Введите номер машины: A123
Действие (enter/exit/lost/next/quit): exit
Выезд: A123
Машина A123 выехала в момент 2.3. Стоял 3.0 ч., стоимость 150.0 руб. Свободно мест: 3

Введите номер машины: quit
Действие (enter/exit/lost/next/quit): quit
Логи сохранены в parking.log
```
